盛世源码—国内优秀的微信公众服务平台新版微酒店正式上线

2014-03-27

尊敬的盛世源码—国内优秀的微信公众服务平台用户，大家好！

     新版微酒店正式上线啦！ 您还在为多人不能同时提交订单，而造成的客人流失发愁吗？您还在为一直安排工作人员不停的更新库存，而长时间的盯着屏幕吗？您还在为客人不能在线付款，为要不要长时间留房而纠结吗？ 不用担心，这一切的一切，都已得到解决！ 

     具体更新功能如下： 

    1. 增加支付功能 

    2. 增加房间库存 

    3. 我的订单管理的优化 

    4. 用户体验优化，主要有： 

    4.1. 缩短了订单流程 

    4.2. 酒店首页做了全面优化 

    4.3. 把原有的点击房间页做了全面优化 

    4.4. 预订页的全面优化 上线后，为了能够让老版微酒店的用户也用上全新的微酒店，盛世源码—国内优秀的微信公众服务平台团队将对老数据进行迁移，具体如下： 

    ● 之前所有的订单的支付方式变更为：前台支付 未支付； 

    ● 过了入住日期的订单，状态依然保留“已确认”的状态； 

    ● 过了入住日期且在“已确认”状态下的订单，我们给商家预留24小时的处理时间，可以处理为“已退房”或“订单作废”，否则我们将自动处理成“已退房”，订单的支付方式变更为：前台支付 已支付； 

    ● 之前的订单做退房处理时，因之前没有离店日期，故离店日期在入住日期的基础上+1； 附上微酒店的设计稿请大家欣赏下：


 



是不是觉得高端大气上档次呢？那赶快跟着我们一起去后台配置吧？

1、洒店的效果图设置，请到编辑酒店信息的商家图片中添加，图片高度建议400px；

2、基础设施的设置直击用户最关心的上网、停车、就餐等问题；

3、房间总数的设置规则：0为不限制允许被预订的房间数，建议立即更改为实际可被预订的房间数；

4、房间的配套设置设置，在新建或编辑房间时，勾选房间已有的配套设施；
