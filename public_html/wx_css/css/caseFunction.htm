  • 微调研
    微调研是基于盛世源码平台而展现出的一种新的在线调研应用方式，主要以问卷调查的方式。在此模块，商家可以通过发起微调研来了解用户需求，用户心态，从而制定更好的发展模式。

    微调研涉及家用电器、数码产品、美食、房产、汽车快速消费品、游戏等多个行业，为商家和企业了解用户的需求到来的极大的帮助。


    商家在此板块中可以设置服务改进等方面的调研活动，设置好需要调研的题目，让用户进行选择，通过调研结果改进对应的服务。


    商家可以设置参加调研后的奖品和获取调研者的信息，通过微调研商家或企业可以看到参与调研的用户信息，变相的为商家和企业增加了商机。并可以对调研活动进行统计。
