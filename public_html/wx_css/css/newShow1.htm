“万人空巷”——盛世源码助力全国创业咖啡联盟昆明年会

2014-03-04

      提起创咖联盟，大家应该不是很陌生，因为创咖联盟的诞生颇具传奇色彩：


      2012年5月27日，中央电视台《新闻联播》头条第一次将镜头对准了车库咖啡，第一次把咖啡馆作为一个创业服务机构。


      2012年5月18日，董建强在微博上提议建立中国创业咖啡联盟，这一提议很快得到同仁们的认可，全国20多个地方的创业咖啡馆积极响应加入到创业咖啡联盟的筹备工作中来。


      2013年9月1日，全国已经有20个省市80余家创业服务的咖啡馆开业，极大地带动了当地的创业活力和服务水平。


      2014年1月3日，盛世源码平台与全国创业咖啡联盟达成战略合作，和中创盟成立联合公司，致力打造创业咖啡行业最全面、最成熟的盛世源码行业解决方案，打造线上线下深度结合的创业主题咖啡会员平台、交流平台和活动平台，并将全力推动全国创业主题咖啡的快速发展。



      2014年3月8—9日，中国创业服务峰会、中国创业咖啡联盟年会在云南昆明泰丽国际酒店盛大举行。本次年会活动由盛世源码产品技术全力支持，活动一开始就已经吸引过万人的参与，一时间形成了“万人空巷”的火爆局势。这个结果让董总倍感意外，却是盛世源码预料之中的结果！


年会活动

中创盟微官网


      本次活动的火爆是偶然也是必然，转发有奖只是盛世源码产品中众多功能中的一个，商家可以在微官网里设置转发有奖活动，用户通过关键词回复参加活动，填写相关信息，然后将页面转发给还有或者发到朋友圈，系统可以自动统计好友点击页面的次数，每个好友只能点击一次。看似一个很小的功能，但是用户体验效果却是极好的，现在盛世源码营销模块中的功能就已经多达十余种，已能完全满足商家的各种需求。当然，盛世源码也在不断努力开发新的功能，让商家用得好，用得爽！


      新的展示传播模式，给人无限想象空间


      如果感兴趣的朋友可以扫一扫关注创咖联盟微信二维码，我们可以发现，创咖联盟年会的整个活动流程我们都可以从微官网中获取，包括会议流程、会议议程、会议嘉宾、甚至你不用去现场就可以报名注册参加，整套活动流程，我们都可以在线上完成。


      现在很多商家都建立微信公众平台，但是如何有效的将信息传播出去却是个大问题，营销活动结合微信公众平台，用户关注商家微信公众号后参与活动，在线下活动时把现场顾客全都转化成粉丝，然后再通过有奖转发等一些营销活动将粉丝的粉丝发展成粉丝，商家不再需要绞尽脑汁去找粉丝了，因为已经有人帮你找了，我们可以预见，这一新玩法还将颠覆微信公众号活动涨粉模式！
